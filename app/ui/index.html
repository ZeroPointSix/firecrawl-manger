概览（/admin/stats）

快速确认 Key/Client 状态；需要先连接 Admin Token。
尚未加载。

最小使用流程

UI 仅调用既有控制面 API（/admin/*），不引入额外后端语义。
1) 点击右上角「连接」，输入 FCAM_ADMIN_TOKEN（仅内存保存）。
2) 在「API Keys」创建/测试 Key（入库后只展示 masked）。
3) 在「Clients」创建 Client，复制一次性 token，业务方用该 token 调用 /api/*。

数据面自检（/api/scrape）

端到端验证：Client Token → 网关鉴权/治理 → Key 池选择 → 上游 scrape。若本实例关闭了 data plane，会返回 404。
不会持久化保存；仅用于本次自检。

帮助

                  
安全与边界说明（请务必阅读）。

                

              

              

                

                  
定位：本 WebUI 是静态页面，仅用于调用现有 /admin/*。

                  

                    安全建议：
                    
    
                      
  • 生产建议关闭 /docs，并仅内网暴露控制面端口。
    • 
                      
  • 
                        本 UI 支持可选 token 持久化（同标签页/本机），并可配置过期；仅建议在本机/内网使用，且定期轮换 token。
                      
    • 
                      
  • 删除操作均为“软禁用”，避免影响历史日志追溯。
    • 
                    

                  

                


                

                  
命令行自检（示例，不要在工单/群里粘贴明文 token）：

                  
# PowerShell
$h=@{ Authorization = "Bearer <ADMIN_TOKEN>" }
Invoke-RestMethod -Method GET -Uri "<ORIGIN>/admin/stats" -Headers $h
